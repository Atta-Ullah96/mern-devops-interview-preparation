# HTML Module

This folder contains my HTML interview preparation notes.

## Learning Goal

The goal of this module is to understand HTML from the beginning: why it was created, how it structures web pages, how browsers parse it, and how HTML supports SEO, accessibility, forms, and modern web applications.

For every topic:

1. Read the explanation.

4. Open the related example file in the browser.
5. Push the notes to GitHub.
