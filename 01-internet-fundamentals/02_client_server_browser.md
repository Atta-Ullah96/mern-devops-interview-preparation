 What is Client?

A client is the device, browser, or frontend application that sends requests to the server.

Examples of clients:

Browser
Mobile app
React app
Next.js frontend
Postman


 What is Server?

A server is a computer or backend application that receives requests, processes them, talks to the database if needed, and sends responses back to the client.



Example:

When a user clicks the login button, the client sends an HTTP request with email and password to the server. The server validates the user, checks the database, creates a session, and sends a response back to the client.

What is Browser ? 
A browser is a client application that helps users access websites.

Examples:

Chrome
Firefox
Safari
Edge

The browser can:

Send HTTP requests
Receive HTTP responses
Read HTML
Apply CSS
Run JavaScript
Create DOM
Render web pages
Store cookies and cache
