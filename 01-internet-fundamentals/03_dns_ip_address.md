What is IP Address?

An IP address is the real address of a device or server on the internet.

Humans use names like:

google.com

But computers communicate using IP addresses.

Example:

google.com = human-friendly name
142.250.xxx.xxx = computer-friendly address

What is DNS?

DNS stands for Domain Name System.

DNS converts domain names into IP addresses.

Example:

google.com → 142.250.xxx.xxx

We need DNS because humans remember names easily, but computers communicate using IP addresses.

Interview answer:

DNS is like the phonebook of the internet. It converts domain names like google.com into IP addresses so the browser can connect to the correct server.
