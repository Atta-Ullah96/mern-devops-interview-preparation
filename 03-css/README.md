# CSS Interview Preparation

This folder contains CSS notes for MERN + DevOps interview preparation.

## Topics covered in this module

1. CSS history and why CSS was created
2. Inline CSS, internal CSS, and external CSS
3. CSS selectors
4. Specificity
5. Inheritance
6. Cascade vs specificity vs inheritance
7. Practical examples
8. Interview questions

## Goal

The goal is not only to memorize CSS syntax, but to understand why CSS exists, how the browser applies CSS rules, and how to explain CSS concepts clearly in interviews.

## Recommended learning flow

1. Read the notes.
2. Open the examples in browser.
3. Change values and observe behavior.
4. Answer the interview questions without looking.
5. Push your notes to GitHub.
